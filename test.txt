Тестовое задание:
Сделать приложение на Node.js и Angular 1

1. На странице должно быть поле для ввода текста (input).
2. Рядом с полем список в который должно подгружатся с сервера автодополнение (как при поиске в google) в зависимости от значения введенного в поле
(при изменении текста в поле отправляется XHR запрос на сервер для получения автодополнения по новому тексту)
3. Список реализовать через Angular директиву (которая принимает значение input как параметр)
4. Страничку оформить с использованием Bootstrap фреймворка (http://getbootstrap.com)
5. Результат залить на https://github.com/ (это важно)
